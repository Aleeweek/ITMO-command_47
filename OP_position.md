# Позиция дисциплины в образовательной программе для подготовки специалистов в области ИИ

## Пререквезиты (предшествующие) дисциплины
Изучение дисциплины базируется на знаниях, полученных обучающимися при освоении следующих дисциплин:
* дискретная математика (BD-3.2, Б);
* алгоритмизация и программирование (BD-3.1, Б);
* современные высокоуровневые языки программирования (BD-3.1, Б);
* компьютерные сети (BD-3.2, Б);
* операционные системы (BD-3.2, Б).
 
На основе которых студент должен иметь следующие **знания**:
* основные понятия дискретной математики по темам: теория множеств, теория отношений и теории графов;
* основы реляционной алгебры (операции над отношениями);
* базовые принципы алгоритмизации (линейные, ветвящиеся и циклические структуры);
* основные структуры данных (массивы, записи, деревья);
* этапы проектирования пользовательских приложений;
* сетевую модель стека протоколов TCP/IP, протоколбезопасной передачи данных https;
* основные средства, предоставляемые современными операционными системами прикладным программам для решения системных и пользовательских задач;
* базовые понятия и парадигмы современных языков программирования высокого уровня.

**Умения**:
* корректно применять основные алгоритмы для выполнения реляционных операций над отношениями;
* решать задачи математического программирования на основе аналитических методов;
* формализовать предметную область с использованием математических моделей;
* разрабатывать алгоритмы средней сложности на языке программирования;
* работать с составными типами данных (структуры, объекты);
* анализировать вычислительную сложность алгоритмов;
* организовать сетевые взаимодействия и передачу данных в рамках создания систем искусственного интеллекта;
* выбирать операционную систему и ее параметры с учетом требований к развертыванию и сопровождению моделей искусственного интеллекта в среде эксплуатации;
* использовать современные языки программирования для разработки программного обеспечения.

**Навыки**:
* работы с современными средствами проектирования пользовательских приложений;
* решения логических задач из дискретной математики;
* программирования на одном из языков;
* обработки структур данных в программах;
* работы с основными средствами и методами, используемыми в индустрии ИТ для поддержания сетевой инфраструктуры промышленных систем искусственного интеллекта;
* реализации скриптов и настройки операционной системы для автоматизации запуска, мониторинга и устойчивой работы сервисов искусственного интеллекта в среде эксплуатации;
* разработки, тестирования и отладки программ с использованием современных языков программирования.
 
## Кореквизиты (смежные) дисциплины
Освоение данной дисциплины проходит параллельно со следующими дисциплинами:
* машинное обучение (BD-2.2, С);
* технологии проектной деятельности (BD-2.2, С);
* интеллектуальный анализ данных (BD-2.2, С);
* технологическая практика (BD-2.2, С).
 
## Постреквизиты (последующие) дисциплины
Освоение данной дисциплины является теоретической и практической базой для следующих дисциплин и практик:
* практикум по цифровому проектированию (BD-3.1, С);
* большие данные и облачные технологии (BD-3.2, П);
* управление информационными ресурсами (BD-2.2, С);
* проектно-технологическая практика (BD-3.1 / BD-3.2, П);
* основы защиты данных в интеллектуальных системах (BD-2.2, С);
* выполнение и защита ВКР (BD-3.1 / BD-3.2, П).
