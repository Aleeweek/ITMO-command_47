# Описание
Данный репозиторий содержит, открытую для использования, рабочую программу дисциплины (РПД). Она полученна [авторами](README.md#Об-авторах) результате прохождения курса ДПО "Преподавание современного ИИ на топ-уровне: системное видение" проводимого ИТМО (г. Санкт-Петербург) и базируется на компетентностно-ролевой модели [(КРМ-3.0)](https://docs.google.com/spreadsheets/d/12P2yGZSkDgTELrZLS0HHdVlQ6k4TsJB2/edit?usp=sharing&ouid=102532098294721562437&rtpof=true&sd=true)

# 1. Основное описание РПД
### Название дисциплины
Инфраструктура баз данных для задач ИИ

### Роли специаллистов в области ИИ
Способствует для формирования следующих **ролей** специалистов
- *Data Engineer (Инженер по данным)*
- *Data Architect (Архитектор данных)*

### Цель и задачи
**Целью** дисциплины является формирование у студентов системы теоретических знаний и практических инженерных компетенций в области проектирования, развертывания и эксплуатации специализированных инфраструктур хранения и обработки данных, используемых на всех этапах жизненного цикла систем искусственного интеллекта — от сбора и предобработки структурированных и неструктурированных данных до организации эффективного векторного поиска, гибридных RAG-архитектур и масштабируемой аналитики.

**Основные задачи**
1. Сформировать компетенции в области разработки, отладки и тестирования прикладных решений задач баз данных с элементами ИИ.
2. Освоить навыки создания и использования баз данных NoSQL в хранилищах данных на уровне применения наиболее известных подходов и технологий каждого класса хранилищ.
3. Сформировать навыки применения инструментов интеграции данных, научиться определять требования к наборам данных для решения задач машинного обучения, проводить разметку и анализ наборов данных, оценивать качество данных, обеспечивать непрерывную интеграцию данных.
