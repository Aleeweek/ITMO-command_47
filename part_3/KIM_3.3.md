# КИМ 3.3. Лабораторная работа. Развёртывание мини-кластера: Hadoop + Kafka + PostgreSQL (Docker)

Тип оценочного средства (предмет контроля): защита отчета

Перечень контролируемых тем (модулей): Модуль 3

### Цель 
научиться развертывать локальную интегрированную среду (контейнеры Docker Compose) с HDFS, Kafka и PostgreSQL, проверять взаимодействие компонентов.
### Задачи
1.	Подготовка окружения: Docker + Docker Compose.
2.	Создание docker-compose.yml с сервисами: zookeeper, kafka, namenode/datanode (HDFS), postgres.
3.	Конфигурация томов, сетей и переменных среды.
4.	Запуск и проверка статуса всех сервисов.
5.	Простейшая интеграция: записать файл в HDFS, отправить сообщение в Kafka, вставить запись в PostgreSQL.
6.	Сбор логов и создание отчёта о настройке.
