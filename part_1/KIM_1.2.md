# КИМ 1.2. Манипулирование данными (DML) для подготовки аналитических датасетов: фильтрация, сортировка, соединения и т.д.

Тип оценочного средства (предмет контроля): защита отчета

Перечень контролируемых тем (модулей): Модуль 1

### Цель

Формирование у обучающегося навыков составления и оптимизации SQL-запросов для выборки, фильтрации, агрегации и соединения данных с целью подготовки аналитических датасетов на основе разработанной в предыдущей работе реляционной схемы БД.

### Задачи

Выполнение задач представленной лабораторной работы происходит с использованием схемы БД, разработанной в рамках выполнения [КИМ 1.1. Лабораторная работа](./KIM_1.1.md).

1. Составить запросы с использованием фильтрации (WHERE), сортировки (ORDER BY) и ограничения выборки (LIMIT/OFFSET) для получения целевых подмножеств данных.
2. Реализовать запросы с группировкой (GROUP BY), запросы присутствия данных в выборках и агрегатными функциями (COUNT, SUM, AVG, MIN, MAX) для получения сводной аналитической информации.
3. Выполнить запросы с различными типами соединений (INNER, LEFT, RIGHT, FULL JOIN) для объединения данных из нескольких отношений.
4. Применить вложенные подзапросы и операции над множествами (UNION, INTERSECT, EXCEPT) для формирования сложных аналитических выборок.
5. Обработать неопределённые значения (NULL) с использованием COALESCE, NULLIF и трёхзначной логики SQL для корректного формирования набора данных.

Сумел ответить на дополнительные вопросы, связанные не только с процессом выполнения лабораторной работы, но и с пониманием совершенных действий и решенных задач;

Сумел ответить на вопросы, связанные с процессом выполнения лабораторной работы;
